Ejemplo de cifrado AES


El algoritmo de cifrado AES (Advanced Encryption Standard) es una técnica de cifrado de clave simétrica con una matríz de estado de 16 bytes (128 bits) y con claves de 16, 24 y 32 caracteres, (128, 192 y 256 bits respectivamente).



Al dar clic en el botón "Comenzar", se le pedirá que haga lo siguiente:

Escribir o pegar el texto a cifrar

Seleccionar el tipo de cifrado AES (clave de 16, 24 o 32 caracteres)

Escribir la clave para cifrar el texto

Después de obtener el mensaje cifrado, tendrás que:

Escribir la misma clave que usaste para cifrar

Descargar el archivo .txt descifrado. Este contendrá el mensaje original
